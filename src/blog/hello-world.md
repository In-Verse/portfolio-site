I built this website with the [starter eleventy base blog](https://github.com/11ty/eleventy-base-blog). Its been a couple years since I had a website online. Ironically, I made two websites in the last year for other people. I made a Jekyll website for a friend's [art portfolio](https://xanderrabii.com/) and a geocities-like website for the [de-trashing club](https://pickuppigeons.nekoweb.org/) I am part of. 

I decided to go with the base blog, but it had so many plugins that I decided to minimize it. By reducing the features, I hope that I can better understand each one. When I add RSS to my site, I will know each step I took and the files made along the way.

## Inspirations

One of the struggles of making a personal website is making it feel *representative* to you.

As someone who loves the terminal, I wanted it to feel similar to terminal user interface (TUI). A major inspiration was the website of Arthur Wallendorff. Like a good TUI, you can navigate the website using the keyboard.

![Wallen Art](/assets/images/hello-world/wallenart.png "TUI portfolio website of Arthur Wallendorff")

Other inspirations:
- [Hugo Risotto theme](https://themes.gohugo.io/themes/risotto/)
  - Inspired by terminal ricing with a prompt at top
- [Hugo Shell theme](https://github.com/Yukuro/hugo-theme-shell)
  - Looks like a terminal prompt
- [Orhun's Blog](https://orhun.dev/)
  - Formatted like a man page
